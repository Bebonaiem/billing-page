3. Push tag: git push origin v1.0.0
4. Deploy website/ folder to GitHub Pages or your domain

OPTION 2: Your Website Only
────────────────────────────
1. Upload website/ folder to your hosting
2. Point domain to website folder
3. Users download ZIP, extract, run installer
4. No GitHub needed (but recommended for collaboration)

OPTION 3: Just Use Installer
────────────────────────────
1. Upload entire billing-system/ to your server
2. Users open http://yourdomain/installer/install.php
3. Web wizard handles setup
4. Ready to use

⚡ QUICK COMMANDS:
==================

Setup GitHub:
$ git init
$ git add .
$ git commit -m "Initial: BillingHub v1.0.0"
$ git remote add origin https://github.com/YOU/billinghub.git
$ git push -u origin main

Create Release (this auto-packages everything):
$ git tag -a v1.0.0 -m "Release v1.0.0"
$ git push origin v1.0.0

Test Installer:
$ unzip billinghub-1.0.0.zip
$ cd billinghub-1.0.0
$ php -S localhost:8000
→ Open: http://localhost:8000/installer/install.php
